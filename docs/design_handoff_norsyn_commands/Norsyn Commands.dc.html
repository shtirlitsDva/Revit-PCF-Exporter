Norsyn Commands · Revit dockable pane
Three directions for the command palette
All three mockups are live — type in the search box, switch Category / Source / A–Z, click a header to collapse it, and tap the ★ stars to pin commands. The floating Favorites palette (drag it by its header) updates instantly. Shown at a ~300px docked width in Revit’s dark theme.
Jump to: 1a Dense list  ·  1b Icon grid  ·  1c Command palette
1a A · Dense list
Tightest rows — closest to a native Revit / Visual Studio tool window, best raw scan-density. Scales to 80+ via live search plus collapse. The fold tension is eased by an always-visible Expand all and by auto-expanding every group the moment you start typing.

1c C · Search-first command palette
One big field, results below, nothing ever folded — optimised for “type three letters, hit Enter.” It resolves the hide-things tension by never hiding: categories are just quiet dividers. Scales effectively without limit because you rarely scroll, you filter. Shown pre-filtered on “ins” — clear it to see all.
